# 학온(HAGON) 데이터베이스 설계

Supabase PostgreSQL 테이블 구조 및 관계 정의

---

## 현재 테이블 (MVP)

### contracts
계약서 핵심 데이터 저장

| Column | Type | Description |
|--------|------|-------------|
| id | UUID (PK) | 고유 식별자 |
| user_id | UUID (FK) | auth.users 참조 |
| contract_data | JSONB | 마법사 입력 전체 데이터 |
| status | TEXT | draft, completed, signed |
| created_at | TIMESTAMPTZ | 생성일시 |
| updated_at | TIMESTAMPTZ | 수정일시 |

> **Notes:** RLS: 자기 계약서만 CRUD 가능. 인덱스: user_id, status, created_at DESC

### partners
파트너사 관리 (플랜, 쿼터, 인증키)

| Column | Type | Description |
|--------|------|-------------|
| id | UUID (PK) | 고유 식별자 |
| name | TEXT | 파트너사 명칭 |
| api_key | TEXT | 인증용 키 |

### partner_sessions
세션 토큰 (10분 만료, 일회성)

| Column | Type | Description |
|--------|------|-------------|
| id | UUID (PK) | 고유 식별자 |
| partner_id | UUID (FK) | partners 참조 |
| token | TEXT | 일회성 토큰 |
| expires_at | TIMESTAMPTZ | 만료 시간 |

### partner_contracts
파트너 경유 계약 추적

| Column | Type | Description |
|--------|------|-------------|
| id | UUID (PK) | 고유 식별자 |
| contract_id | UUID (FK) | contracts 참조 |
| partner_id | UUID (FK) | partners 참조 |

## 고도화 예정 테이블 (Phase 2)

### organizations
법인 정보 관리

| Column | Type | Description |
|--------|------|-------------|
| id | UUID (PK) | 고유 식별자 |
| name | TEXT | 법인명 |
| biz_number | TEXT | 사업자번호 |
| plan | TEXT | 구독 플랜 |

### academies
학원 정보 관리

| Column | Type | Description |
|--------|------|-------------|
| id | UUID (PK) | 고유 식별자 |
| org_id | UUID (FK) | organizations 참조 |
| name | TEXT | 학원명 |
| address | TEXT | 주소 |

### user_profiles
사용자 프로필 및 소속 관리

| Column | Type | Description |
|--------|------|-------------|
| id | UUID (FK) | auth.users 참조 |
| org_id | UUID (FK) | organizations 참조 |
| academy_id | UUID (FK) | academies 참조 |
| role | TEXT | admin, manager, user |

### subscriptions
구독 및 결제 관리

| Column | Type | Description |
|--------|------|-------------|
| id | UUID (PK) | 고유 식별자 |
| org_id | UUID (FK) | organizations 참조 |
| status | TEXT | active, expired, canceled |
| expires_at | TIMESTAMPTZ | 만료일 |

### contract_templates
계약서 템플릿 버전 관리

| Column | Type | Description |
|--------|------|-------------|
| id | UUID (PK) | 고유 식별자 |
| type | TEXT | 계약 유형 |
| schema | JSONB | 필드 정의 스키마 |

### notifications
시스템 알림

| Column | Type | Description |
|--------|------|-------------|
| id | UUID (PK) | 고유 식별자 |
| user_id | UUID (FK) | auth.users 참조 |
| title | TEXT | 제목 |
| read_at | TIMESTAMPTZ | 읽은 시간 |

### audit_logs
감사 로그

| Column | Type | Description |
|--------|------|-------------|
| id | UUID (PK) | 고유 식별자 |
| user_id | UUID (FK) | 수행자 |
| action | TEXT | 수행 작업 |
| metadata | JSONB | 상세 변경 내역 |
