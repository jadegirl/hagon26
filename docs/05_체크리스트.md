### 서랍 01: 채용 및 계약 단계 (8종)
- [ ] 강사근로계약서 — 기존 wizard type-a 통합/마이그레이션
- [ ] 프리랜서강사계약서 — 배분율, 해지 조건
- [ ] 전속강사계약서 — 전속 범위, 겸업 금지, 위약금
- [ ] 근로계약서(행정직) — 사무·데스크 직원용
- [ ] 수습·시용 기간 합의서 — 시용 기간, 평가 기준
- [ ] 연봉계약서 — 연봉 총액, 상여금
- [ ] 개인정보 수집·이용 동의서 — 수집 항목, 보유 기간
- [ ] 성범죄경력조회 동의서 — 학원법 제22조 의무

### 서랍 02: 근태 및 복무 관리 (6종)
- [ ] 연차·휴가 신청서 — 휴가 종류, 업무 대행자
- [ ] 연장·휴일근로 동의서 — 주말 수업·보강 동의
- [ ] 외출·조퇴·지각 확인서 — 근태 기록 증빙
- [ ] 경위서 — 발생 경위, 재발 방지 약속
- [ ] 보안유지·비밀준수 서약서 — 보호 대상, 위반 시 손해배상
- [ ] 직장 내 괴롭힘·성희롱 예방 교육 확인서 — 법정 의무 교육 이수

### 서랍 03: 퇴직 및 계약 종료 (6종)
- [ ] 사직서 — 최종 근무일, 퇴사 사유, 반납 확인

- [ ] 경력·재직증명서 발급 신청서 — 퇴사 후 행정 서비스
- [ ] 인수인계 확인서 — 업무·학생 데이터 이관 확인
