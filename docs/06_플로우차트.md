# 학온(HAGON) 전체 플로우차트

서비스 전체 흐름을 Phase별로 정리한 문서입니다.

---

## 전체 서비스 흐름 (조감도)

```
┌─────────────────────────────────────────────────────────────────┐
│                        학온(HAGON) 플랫폼                         │
├─────────────┬──────────────┬──────────────┬─────────────────────┤
│  🏠 공개 영역  │  📋 사용자 영역  │  🛡️ 관리자 영역  │  🔗 파트너 연동 영역  │
│             │              │  (Phase 2)   │                     │
│  랜딩 페이지  │  대시보드      │  총관리자      │  파트너 SSO          │
│  로그인      │  계약서 마법사  │  학원관리자     │  Webhook 콜백       │
│             │  전자서명      │  구독/요금     │  세션 토큰           │
│             │  계약서 조회   │  사용자 관리    │  취소 처리           │
└─────────────┴──────────────┴──────────────┴─────────────────────┘
```

```

┌──────────────┐
```

## 기존 MVP — 파트너 연동 플로우

학원관리 프로그램(TeamPro 등)에서 학온으로 연동되는 흐름입니다.

```
┌────────────────────────┐
│  🏢 파트너 앱 (TeamPro)   │
│                        │
│  "전자계약 시작" 버튼 클릭  │
└────────────┬───────────┘
             │ API 호출
             ▼
┌────────────────────────────────────────┐
│  🔑 세션 토큰 발급                        │
│  POST /api/partner/session             │
│                                        │
│  파트너 API Key + HMAC-SHA256 서명 검증    │
│  → 10분 만료 일회성 토큰 발급              │
└────────────┬───────────────────────────┘
             │ 토큰 포함 리다이렉트
             ▼
┌────────────────────────────────────────┐
│  🚀 파트너 런칭 (/partner/launch)        │
│                                        │
│  1. 토큰 검증 (GET /api/partner/verify)  │
│  2. SSO 자동 로그인 (/api/partner/sso)    │
│  3. 대시보드로 자동 이동                    │
└────────────┬───────────────────────────┘
             │ 자동 로그인 완료
             ▼
┌─────────────────────────┐
│  📋 대시보드 (/dashboard)  │
│                         │
│  (이후 일반 사용자와        │
│   동일한 계약서 작성 플로우)  │
└────────────┬────────────┘
             │ 계약서 작성 → 서명 완료
             ▼
┌────────────────────────────────────────┐
│  📡 계약 완료 콜백                        │
│  POST /api/partner/callback            │
│                                        │
│  파트너사에 계약 완료 알림 전송             │
│  (계약ID, 상태, 완료일시)                  │
└────────────────────────────────────────┘
```

**취소 시 흐름:**

```
계약 진행 중 취소 → POST /api/partner/cancel → 파트너사에 취소 알림
```

## Phase 1 — MVP 고도화 플로우

기존 MVP의 품질을 높이는 단계입니다. 새로운 페이지 추가 없이 기존 화면을 개선합니다.

```
┌─────────────────────────────────────────────────────────┐
│                    Phase 1 개선 영역                      │
│                                                         │
│  📝 계약서 마법사 ──→ UI/UX 리뉴얼 (템플릿 v7 반영)        │
│     └── 미리보기 ──→ 레이아웃 교정 (조항 순서, 급여 테이블)   │
│                                                         │
│  📋 대시보드 ────→ UI 개선 (통계 카드, 필터 강화)            │
│                                                         │
│  ✍️ 전자서명 ────→ 플로우 안정화                            │
│                                                         │
│  📄 PDF ────────→ 품질 개선 (서명란 정리)                   │
│                                                         │
│  ⚠️ 에러 처리 ──→ 전체 에러 핸들링 강화                     │
└─────────────────────────────────────────────────────────┘
```

**Phase 1 완료 후 변경되는 플로우 (차이점):**

| 화면 | 변경 전 (MVP) | 변경 후 (Phase 1) |
|------|-------------|-----------------|
| 계약서 마법사 | 기본 UI | 템플릿 v7 기반 리뉴얼 UI |
| 미리보기 | 기본 레이아웃 | 조항 순서 정리 + 급여 테이블 개선 |
| 대시보드 | 기본 목록 | 통계 카드 + 고급 필터 |
| 전자서명 | 기본 서명 | 안정화된 서명 플로우 |
| PDF | 기본 출력 | 고품질 PDF + 서명란 정리 |

## Phase 2 — 관리 시스템 플로우

회원 체계가 확장되고, 총관리자/학원관리자 역할이 추가됩니다.

### 회원 계층 구조

```
┌──────────────────────────────────────────┐
│  👑 슈퍼관리자 (학온 운영팀)                 │
│  - 시스템 전체 관리                         │
│  - 모든 학원/계약/사용자 조회                 │
│                                          │
│     └── 🏢 법인 관리자 (학원 법인 대표)       │
│         - 소속 법인 내 학원 관리              │
│         - 소속 법인 내 계약서/인사서류 조회     │
│                                          │
│            └── 🏫 학원 관리자 (개별 학원 원장) │
│                - 소속 학원 계약서 관리         │
│                - 소속 학원 강사 관리           │
│                                          │
│               └── 👨‍🏫 강사 (서명/열람 전용)  │
│                   - 본인 계약서 열람           │
│                   - 전자서명                  │
└──────────────────────────────────────────┘
```

### 총관리자 플로우

## 화면 간 이동 경로 요약

### 공개 → 인증

| 출발 | 도착 | 조건 |
|------|------|------|
| `/` (랜딩) | `/login` | "시작하기" 클릭 |
| `/login` | `/dashboard` | 로그인 성공 (일반/학원관리자) |
| `/login` | `/admin` | 로그인 성공 (슈퍼관리자, Phase 2) |

### 대시보드 → 계약서

| 출발 | 도착 | 조건 |
|------|------|------|
| `/dashboard` | `/wizard/type-a/step-1` | "새 계약서" 클릭 |
| `/wizard/type-a/step-N` | `/wizard/type-a/step-N+1` | 다음 단계 |
| `/wizard/type-a/preview` | `/wizard/type-a/complete` | 계약서 저장 |
| `/dashboard` | `/view/[id]` | 계약서 클릭 (조회) |

### 전자서명

| 출발 | 도착 | 조건 |
|------|------|------|
| 카카오톡/URL | `/sign/[id]` | 강사가 서명 링크 클릭 |
| `/sign/[id]` | `/sign/[id]/success` | 서명 완료 |

### 파트너 연동

| 출발 | 도착 | 조건 |
|------|------|------|
| 파트너 앱 | `/partner/launch` | SSO 토큰 포함 리다이렉트 |
| `/partner/launch` | `/dashboard` | 자동 로그인 완료 |

### Phase 2 관리자

| 출발 | 도착 | 조건 |
|------|------|------|
| `/admin` | `/admin/academies` | 학원 관리 메뉴 |
| `/admin` | `/admin/subscriptions` | 구독 관리 메뉴 |
| `/admin` | `/admin/users` | 사용자 관리 메뉴 |
| `/academy/[id]/dashboard` | `/academy/[id]/contracts` | 계약서 관리 메뉴 |
| `/academy/[id]/dashboard` | `/academy/[id]/staff` | 강사 관리 메뉴 |
