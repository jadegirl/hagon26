# 학온(HAGON) 전체 화면 구조도

프로젝트의 주요 페이지 및 API 엔드포인트 구조입니다.

---

## 공개 페이지

| 경로 | 페이지명 | 설명 | 상태 |
|------|---------|------|------|
| `/` | 랜딩 페이지 | 서비스 소개, 기능, 가격, 리스크 진단 | ✅ 완료 |
| `/login` | 로그인/회원가입 | 사용자 인증 및 계정 생성 | ✅ 완료 |

---

## 사용자 페이지

| 경로 | 페이지명 | 설명 | 상태 |
|------|---------|------|------|
| `/dashboard` | 대시보드 | 계약서 목록, 진행중/완료 탭, 검색, 카카오 공유 | ✅ 완료 |
| `/wizard/type-a/*` | 계약서 마법사 TYPE_A | 정규직 강사 계약서 작성 프로세스 (Step 1~8) | ✅ 완료 |
| `/sign/[id]` | 전자서명 페이지 | 계약 당사자 전자서명 진행 | ✅ 완료 |
| `/view/[id]` | 계약서 조회 | 계약서 내용 확인 (읽기 전용) | ✅ 완료 |

### 마법사 세부 단계
| 단계 | 경로 | 내용 |
|------|------|------|
| Step 1 | `/wizard/type-a/step-1` | 학원 정보 입력 |
| Step 3 | `/wizard/type-a/step-3` | 강사 정보 입력 |
| Step 4 | `/wizard/type-a/step-4` | 계약기간 설정 |
| Step 5 | `/wizard/type-a/step-5` | 근무조건 설정 |
| Step 6 | `/wizard/type-a/step-6` | 급여 설정 (고정급/비율급/시급) |
| Step 7 | `/wizard/type-a/step-7` | 보호조항 (경업금지 등) |
| Step 8 | `/wizard/type-a/step-8` | 특약사항 |
| Preview | `/wizard/type-a/preview` | 계약서 미리보기 |
| Complete | `/wizard/type-a/complete` | 저장 완료 |

## 파트너 연동

| 경로 | 페이지/API | 설명 | 상태 |
|------|-----------|------|------|
| `/partner/launch` | 파트너 런칭 페이지 | SSO 자동 로그인 및 연동 시작 | ✅ 완료 |
| `/api/partner/session` | 세션 생성 API | 파트너 세션 토큰 발급 | ✅ 완료 |
| `/api/partner/verify` | 토큰 검증 API | 발급된 토큰 유효성 확인 | ✅ 완료 |
| `/api/partner/sso` | SSO API | 단일 로그인 처리 | ✅ 완료 |
| `/api/partner/callback` | 계약 완료 콜백 | 계약 완료 시 파트너사 알림 | ✅ 완료 |
| `/api/partner/cancel` | 취소 처리 | 계약 진행 취소 처리 | ✅ 완료 |
