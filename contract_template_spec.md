## 🔢 자동 계산 로직

### 1. 주휴수당 계산 (고정급)

```typescript
function calculateWeeklyHolidayPay(
  baseSalary: number,           // 월 기본급
  weeklyWorkHours: number,      // 주소정근로시간
  workDaysPerWeek: number       // 주 근무일수
): number {
  // 주 15시간 미만이면 주휴수당 없음
  if (weeklyWorkHours < 15) return 0;
  
  // 1일 소정근로시간
  const dailyWorkHours = weeklyWorkHours / workDaysPerWeek;
  
  // 주휴시간 = 1일 소정근로시간
  const weeklyHolidayHours = dailyWorkHours;
  
  // 월소정근로시간 = (주소정 + 주휴) × 4.345
  const monthlyWorkHours = (weeklyWorkHours + weeklyHolidayHours) * 4.345;
  
  // 통상시급 = 기본급 ÷ 월소정근로시간
  const hourlyRate = baseSalary / monthlyWorkHours;
  
  // 월 주휴수당 = 통상시급 × 주휴시간 × 4.345
  const weeklyHolidayPay = hourlyRate * weeklyHolidayHours * 4.345;
  
  return Math.round(weeklyHolidayPay);
}
```

### 2. 통상시급 계산

```typescript
function calculateHourlyRate(
  baseSalary: number,
  weeklyWorkHours: number,
  workDaysPerWeek: number
): number {
  const dailyWorkHours = weeklyWorkHours / workDaysPerWeek;
  const weeklyHolidayHours = weeklyWorkHours >= 15 ? dailyWorkHours : 0;
  const monthlyWorkHours = (weeklyWorkHours + weeklyHolidayHours) * 4.345;
  
  return Math.round(baseSalary / monthlyWorkHours);
}
```

### 3. 고정급 총액 계산

```typescript
function calculateTotalSalaryFixed(salary: ContractSalary): number {
  return (
    salary.base_salary +
    salary.meal_allowance +
    salary.transport_allowance +
    salary.position_allowance +
    salary.other_allowance +
    salary.weekly_holiday_pay +      // 자동계산
    salary.non_compete_pay_fixed
  );
}
```

### 4. 비율제 잠정주휴수당 계산

```typescript
function calculateProvisionalHolidayPay(
  minGuarantee: number,         // 최소보장액
  weeklyWorkHours: number,
  workDaysPerWeek: number
): number {
  if (weeklyWorkHours < 15) return 0;
  
  const dailyWorkHours = weeklyWorkHours / workDaysPerWeek;
  const weeklyHolidayHours = dailyWorkHours;
  const monthlyWorkHours = (weeklyWorkHours + weeklyHolidayHours) * 4.345;
  
  // 최소보장액 기준 통상시급
  const hourlyRate = minGuarantee / monthlyWorkHours;
  
  // 잠정 주휴수당
  return Math.round(hourlyRate * weeklyHolidayHours * 4.345);
}
```

비밀의 범위는 다음 각 호를 포함한다.
1. 학원 운영 및 경영에 관한 정보
2. 학생 명단, 성적, 학부모 연락처 등 개인정보
3. 교재, 커리큘럼, 수업 자료 등 교육 콘텐츠
4. "갑"이 비밀로 지정한 사항

제9조 (경업금지)
"을"은 퇴직 후 다음의 경업금지 의무를 부담한다.
1. 경업금지 기간: 퇴직일로부터 {{경업금지기간}}
2. "갑"의 사업장 소재지로부터 반경 {{경업금지지역}} 이내
3. 경업금지 대가: 본 계약 제5조의 급여에 포함하여 지급
"을"이 경업금지 의무를 위반하여 "갑"에게 손해가 발생한 경우, "을"은 실제 발생한 손해를 배상할 책임이 있다.

제10조 (손해배상)
"을"이 고의 또는 중대한 과실로 "갑"에게 손해를 입힌 경우, "을"은 그 손해를 배상할 책임이 있다.
"갑"은 "을"에게 근로계약 불이행에 대한 위약금 또는 손해배상액을 예정하는 계약을 체결하지 아니한다.

제11조 (기타)
본 계약에서 정하지 아니한 사항은 근로기준법 및 관계 법령에 따른다.
본 계약의 내용을 증명하기 위하여 계약서 2부를 작성하여 "갑"과 "을"이 각각 서명 또는 기명날인 후 각 1부씩 보관한다.

## 🎯 Cursor 작업 요청 사항

### 1단계: 템플릿 컴포넌트 수정
```
src/components/contract/
├── ContractPreview.tsx        # 미리보기 메인 컴포넌트 수정
├── templates/
│   ├── TypeATemplate.tsx      # TYPE_A 전용 템플릿
│   ├── SalaryTableFixed.tsx   # 고정급 급여 테이블
│   ├── SalaryTableRatio.tsx   # 비율제 급여 테이블
│   └── SignatureBlock.tsx     # 서명란 (2열 테이블)
```

### 2단계: 변수 변환 유틸리티
```
src/utils/
├── contractVariables.ts       # 템플릿 변수 매핑
├── salaryCalculations.ts      # 급여 자동계산 (주휴수당 등)
└── dateFormat.ts              # 날짜 포맷팅
```

### 3단계: 수정해야 할 핵심 사항

1. **제목 수정**: "강사근로계약서" → "근 로 계 약 서 / Employment Contract"
2. **조항 순서**: 템플릿 순서대로 재배열
3. **급여 테이블**: 항목별 테이블 형식으로 렌더링
4. **비율제 상세**: 주휴수당 포함 여부 및 정산 방식 명시
5. **서명란**: 2열 테이블 형식으로 수정
